## 📌 Sobre o Projeto

##  Como instalar C# e criar um Windows Forms

### 1. Instalar o Visual Studio

* Baixa o Visual Studio Community (gratuito)
* Executa o instalador

---

### 2. Selecionar o C#

Durante a instalação, marca:

✔ Desenvolvimento para desktop com .NET

---

### 3. Criar projeto Windows Forms

* Abre o Visual Studio
* Clica em **Create a new project**
* Procura por: `Windows Forms App`
* Seleciona e clica em **Next**

---

### 4. Criar interface

Arrasta componentes como:

* Button
* Label
* TextBox

Exemplo de código:

```csharp
private void button1_Click(object sender, EventArgs e)
{
    MessageBox.Show("Olá, mundo!");
}
```
